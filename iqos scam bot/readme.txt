Все настройки находятся в самом начале файла config.

Для работы бота нужен Python3
Так-же нужно установить библиотеки:
pip3 install SimpleQIWI
pip3 install PyTelegramBotApi

Запуск: python3 main.py

Доступ к админке через команду /admin
Чтобы вернутся обратно в главное меню можно написать /start
